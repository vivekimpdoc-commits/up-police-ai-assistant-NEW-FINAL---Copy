# UP Police AI Assistant (Official Portal Assistant)

An advanced, premium AI-powered chatbot designed to provide real-time information strictly from the official Uttar Pradesh Police portal (https://uppolice.gov.in).

## 🚀 Features
- **Official Data Only**: Strictly limited to verified information from the UP Police portal.
- **2026 Ready**: Uses the latest Gemini 2.5 Flash-Lite engine for maximum speed and accuracy.
- **Voice Support**: Full browser-based Text-to-Speech support for English and Hindi.
- **Multimodal**: Supports image uploads for analysis.
- **SOS Mode**: Quick access to emergency dial 112.

## 🛠️ How to Run Locally

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Variables**:
   Create a `.env` file and add your API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

3. **Build & Start**:
   ```bash
   npm run build
   npm start
   ```

## 🌐 Deployment (Render)
This project is optimized for deployment on **Render** as a Web Service.
- **Build Command**: `npm install && npm run build`
- **Start Command**: `npm start`
